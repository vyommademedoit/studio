# **App Name**: Breathe Easy

## Core Features:

- User Authentication: Secure user login and signup with Firebase Authentication (email/password).
- Habit Tracking: Allow users to create, edit, and track daily habits, calculate streaks, and mark habits as good or bad.
- Journaling: Enable users to create, edit, and save journal entries with tags.
- AI-Powered Daily Reflection: Uses a Genkit flow tool to present a single, thought-provoking question daily and saves the response to the journal.
- Journal Analysis: Provides emotional insights from journal entries, displaying overall sentiment, emotional trends, and mental state insights.
- Meditation Player: Play background ambient sounds. Use a timer to play meditation. Implement a main play/pause button
- Resources Page: Display a curated list of mental health support resources with descriptions, contact information and links to websites.

## Style Guidelines:

- Primary color: Soft blue (#B7D9EA) to promote calmness and serenity. Chosen because blues are closely associated with peacefulness.
- Background color: Very light off-white (#FCFCFE). Its desaturated, brighter tone helps give a sense of space.
- Accent color: Light purple (#B3AEE5). Its association with reflection complements the function of the application.
- Headline font: 'Merriweather', a serif font providing an elegant and readable feel.
- Body font: 'Lato', a sans-serif font to give a readable and professional feel.
- Use light and airy icons from the `Iconsax` library. Chosen because it aligns closely with the 'Breathe Easy' aesthetic.
- Components feature rounded corners (0.5rem radius) and subtle shadows to give a soft and modern aesthetic.
- Subtle fade-in animations and smooth transitions on user interactions to create a seamless experience.
